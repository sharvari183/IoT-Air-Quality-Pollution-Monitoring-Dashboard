# 🌍 IoT-Based Air Quality & Pollution Monitoring Dashboard

## Overview

This project is an industry-oriented IoT and AI-powered Air Quality Monitoring Dashboard developed using Python, Streamlit, Machine Learning, and Data Analytics.

The system simulates environmental sensor data, monitors air quality metrics, predicts AQI levels using a machine learning model, and provides real-time visualization through an interactive dashboard.

## Features

- Real-Time Air Quality Monitoring
- AQI Tracking and Analysis
- PM2.5 Monitoring
- Temperature & Humidity Analytics
- AI-Based AQI Prediction
- Smart Alert System
- Environmental Health Score
- Pollution Risk Assessment
- Smart City Zone Monitoring
- Sensor Health Monitoring
- Interactive Data Visualization
- CSV Report Download
